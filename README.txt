# Python App Compiler

GUI wrapper around PyInstaller for building Windows executables 
with metadata, versioning, icon support and portable packaging.

## Demo
![Create Demo](demo_gui.png)

Currently supports Windows builds.

## Features

- Auto-detect main scripts
- Version metadata injection
- .ico support
- One-file or folder build
- Optional launcher creation
- Portable ZIP generation
- Automatic PyInstaller installation



## Installation

pip install pyinstaller
python app_compiler.py
